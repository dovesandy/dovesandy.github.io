## 方法
1. 父容器`display: flex;`
2. 目标容器`margin: auto`

## 代码
```html
<!DOCTYPE html>
<html>
<head>
<meta charset="utf-8">
<title>垂直居中</title>
</head>
<style>
    html,body {
        width: 100%;
        height: 100%;
    }
    body {
        background-color: aqua;
        display: flex;
    }
    div {
        width: 300px;
        height: 300px;
        background-color: red;
        margin: auto;

    }
</style>
<body>
<div>垂直居中</div>
<script>

</script>

</body>
</html>
```
